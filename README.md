# Flipkart E-Commerce Dashboard

A Power BI dashboard built on 20,000 Flipkart product listings.

## Charts included
- Products by Category
- Discount Distribution
- Price Range breakdown
- FK Advantage donut chart
- Top Brands

## Files
- Flipkart_Dashboard.pbix — Power BI file
- flipkart_cleaned_powerbi.csv — Cleaned dataset

## Tools used
- Python (pandas) for data cleaning
- Microsoft Power BI Desktop for visualization
